<h3>Create an Azure Account</h3>

To create a Microsoft Azure account, visit https://azure.microsoft.com/en-us and select the "Sign In" button in the top right corner. Sign in with your Microsoft account or create one if you don't have an account yet.

<img src="https://i.imgur.com/JbAavOs.png" height="100%" width="100%" alt="Win 11 Hardware"/>
</p>


<h3>200$ Free Azure Credits</h3>

After logging into the Azure Portal, select the option for "Start with an Azure Free Trial," which offers $200 worth of free Azure credits. Fill out the required information, and you should be approved!
